Couch Potato



之前打工課業兩頭燒的時候，
看影集是我唯一可以忙裡偷閒的活動，
但久了以後發現脫不了了，
沒事時好像就應該躺在床上，
甚麼也不要做，
抱著筆電看影集就行了。
我喜歡看有人死掉的影集，
但我會怕青蛙蜘蛛老鼠蟑螂，
所以應該不會有暴力傾向，
請大家不要擔心。

Dreamcatcher



在19.20間，
是一個富有想法富有創造力跟執行力的階段，
剛剛能夠真正的為自己負責任。
而身旁也出現了很多的朋友開始在追自己的夢。
有的人以前上課一直在畫課本，
而現在開始在接case幫人在畫小插畫；
有的人無時無刻戴著耳機，
而現在開始可以自己製作音樂，
夢想成為一名DJ；
有的人很愛打扮自己，
每天看到都有不同的樣子，
而現在也開始經營自己的美甲工作室…
這些人為了自己想做的事付出了多少的努力，
我相信讀完以後你也會更有動力，
為自己的未來更進一份力。
